## 탐색기 열기

```
explorer
```

## 현재 경로에서 탐색기 열기

```
explorer .
```

## 주어진 경로에서 탐색기 열기

```
explorer [PATH_NAME]
```

## 문법 사항

- explorer 또는 Explorer.exe 명령어를 쓰는 것 둘 다 상관없이 같은 결과를 기대할 수 있다.
