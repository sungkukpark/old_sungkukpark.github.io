본 포스팅은 Pope Kim의 『셰이더 프로그래밍 입문 (한빛미디어, 2012)』의 Part 1의 Chapter 1에 해당하는 내용(p. 21-45)에 필요한 실습 자료를 제공하는 용도로 작성되었다.

## 개발 환경 설정

- 본 포스팅은 Windows 10 64-bit OS 환경에서 테스트되었다
- 『셰이더 프로그래밍 입문 (한빛미디어, 2012)』의 내용은 Mac OS 환경을 지원하지 않는다
